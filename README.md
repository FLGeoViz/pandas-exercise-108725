## Introduction to Pandas DataFrames – Replicating an article in Markdown

Involved files and their purpose:
- **Notebook file (`.ipynb`)**
    - Contains executable code, Markdown explanations, and cell outputs.
    - Serves as the working and documentation file in Jupyter (interactive and reproducible).
- **HTML file (`.html`)**
    - An exported version of the notebook for viewing in a web browser.
    - Displays code, text, and the outputs present at export time as a static document (no interactivity).
- **Image file (`.jpg`)**
    - A figure embedded in the notebook or in the exported HTML version.
    - Includes alternative text that is shown when the image is unavailable or to support visually impaired users.
